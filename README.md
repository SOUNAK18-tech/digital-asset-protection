# Digital Asset Protection

## 📌 Problem Statement

Sports organizations generate large volumes of digital media that are often misused or redistributed without permission. Tracking and managing such assets is difficult without a centralized system.

## 💡 Solution

This project provides a simple platform to upload, manage, and monitor digital media assets. Users can store files and mark them as "Reported" in case of misuse.

---

## 🚀 Features

* Upload image/video files
* View assets in a responsive grid layout
* Preview media directly in the browser
* Status system: Safe / Reported
* Report misuse with a single click
* Timestamp for each uploaded asset
* Filter assets (All / Safe / Reported)

---

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Backend: Node.js (Express)
* Storage: Local (uploads folder)
* Version Control: Git & GitHub

---

## 📊 Workflow

1. Upload a media file
2. File is stored and displayed
3. Default status is "Safe"
4. User can mark it as "Reported" if misuse is detected
5. Assets can be filtered based on status
